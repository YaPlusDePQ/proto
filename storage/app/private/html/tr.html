TRIGONOMETRIE

Cosinus d’un angle

Dans un triangle rectangle :

cos(Angle) = Adjacent / Hypoténuse

Attention : Le cosinus ne s’applique jamais sur l’angle droit.

Sinus d’un angle

sin(Angle) = Opposé / Hypoténuse

Tangente d’un angle

tan(Angle) = Opposé / Adjacent
